📺 TV Show Search
A simple TV show search app built with vanilla HTML, CSS, and JavaScript, using the TVMaze API.

🔗 [Live Demo](https://adamkirikisla.github.io/show-search-tvmaze/)

Features

* Live search by show name
* Responsive image grid
* Click a show to view full details (poster, summary, status, rating) on a separate page
* Dark mode UI with teal accent theme
* Handles shows with missing images or ratings

Built With

* HTML
* CSS (dark mode, CSS Grid)
* JavaScript (DOM manipulation, Axios, no frameworks)
* TVMaze API<img src="images/favicon.ico" >
